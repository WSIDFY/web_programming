# 계산식을 입력받아 결과를 출력하는 웹 페이지를 작성하라. 식 입력 후 &lt;Enter‘&gt; 키를 치면 수식을 계산하고 결과를 출력한다.

 #### 추가 및 안내 사항

>  &lt;Enter‘&gt;키의 keyCode 값은 13이므로 이벤트 객체 e에 대해, e.keyCode와 13을 비교하면 된다.
